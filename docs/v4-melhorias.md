# Melhorias implementadas na v4

- Painel inteligente de Base de Referências.
- Upload múltiplo de ZIPs resolvidos.
- Salvamento local via IndexedDB.
- Espelhamento opcional em pasta local no Chrome/Edge.
- Score de qualidade por ZIP.
- Marcação: ativo, confiável, legado, ignorado.
- Ranking das referências usadas na geração.
- Logs em tempo real da geração.
- Validação automática da resposta da IA.
- Retry automático quando a feature vier incompleta.
- Diff antes/depois no Analisador.
- Memória local de aprendizado.
- Regras manuais permanentes.
- Exportação/importação de base de conhecimento.
